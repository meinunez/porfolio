Zombicide: Juego de Mesa

Implantación de Aplicaciones Web

Documentación detallada sobre las reglas y componentes del juego de mesa Zombicide. Proyecto desarrollado como parte del curso Implantación de Aplicaciones Web.

Visualizar PDF:

Descripción:

Este proyecto consiste en la implementación de las reglas y componentes del juego de mesa Zombicide, utilizando GitBook para documentar todos los aspectos del juego. Se busca facilitar el acceso a la información clave para los jugadores y entusiastas del juego.

El proceso documentado en GitBook incluye una guía paso a paso para realizar cambios en un repositorio de GitHub. Esto abarca desde la creación de un fork hasta la implementación de mejoras en una rama específica, pasando por la modificación de archivos, la gestión de ramas y la creación de pull requests para revisar y fusionar cambios.

Objetivos:

El objetivo principal fue crear una plataforma de documentación accesible y comprensible que detalle las reglas, componentes y mecánicas del juego, mejorando la experiencia del usuario y optimizando la presentación de la información para una fácil consulta.

Creación y gestión de Vistas: CREATE VIEW

Administración de Sistemas Gestores de Bases de Datos

Documentación detallada sobre la creación, modificación y gestión de vistas en bases de datos. Proyecto desarrollado como parte del curso Administración de Sistemas Gestores de Bases de Datos.

Visualizar PDF:

Descripción:

Este proyecto se centra en la creación y gestión de vistas en bases de datos relacionales, utilizando el comando CREATE VIEW. Una vista es una tabla virtual que proporciona una forma de simplificar consultas complejas y permite presentar la información de forma más accesible y eficiente para los usuarios.

En este proyecto se detalla cómo crear vistas desde cero, cómo actualizarlas, y los beneficios de utilizarlas, como la reutilización de consultas, seguridad al restringir acceso a ciertos datos y mejora en la legibilidad de las consultas SQL.

Objetivos:

El objetivo principal de este proyecto es profundizar en la funcionalidad y uso de las vistas en bases de datos, ofreciendo una guía paso a paso para su creación y gestión. Además, se busca mostrar cómo optimizar el acceso a los datos, mejorar la seguridad y la organización de la información en un entorno de bases de datos.

RAID 10: Seguridad y Alta Disponibilidad

Administración de Sistemas Operativos

Documentación detallada sobre la implementación y características de RAID 10. Proyecto desarrollado como parte del curso Seguridad y Alta Disponibilidad.

Visualizar PDF:

Descripción:

Este proyecto se centra en la implementación de RAID 10, una configuración que combina las ventajas de RAID 1 (duplicación de datos) y RAID 0 (división de datos) para mejorar el rendimiento y la seguridad de los datos. RAID 10 es comúnmente utilizado en sistemas que requieren alta disponibilidad y tolerancia a fallos.

La configuración de RAID 10 permite que los datos estén duplicados y divididos en múltiples discos, proporcionando una combinación de rendimiento y redundancia. En este proyecto se documenta cómo implementar y gestionar RAID 10 en sistemas operativos, con ejemplos prácticos y guías de configuración.

Objetivos:

El objetivo principal de este proyecto es explorar cómo RAID 10 puede aumentar la seguridad y disponibilidad de los datos en un entorno de servidores. Se profundiza en los beneficios de esta configuración, su implementación y cómo gestionar las fallas en discos dentro de una configuración RAID 10.

Análisis de Paquetes con Wireshark: Monitoreo de Red

Seguridad y Alta Disponibilidad

Documentación detallada sobre el uso de Wireshark para capturar y analizar paquetes en una red. Proyecto desarrollado como parte del curso Seguridad y Alta Disponibilidad.

Visualizar PDF:

Descripción:

Este proyecto se centra en el análisis de tráfico de red utilizando la herramienta Wireshark. Wireshark es un analizador de paquetes que permite capturar y examinar los datos que circulan en una red, ofreciendo una visión detallada de su funcionamiento y detectando posibles anomalías.

En este proyecto se detallan los pasos para capturar tráfico de red, filtrar paquetes relevantes y analizar protocolos como TCP, UDP, y HTTP, proporcionando un informe sobre el tráfico capturado y las conclusiones obtenidas a partir del análisis.

Objetivos:

El objetivo principal de este proyecto es aprender a utilizar Wireshark como herramienta de monitoreo de red, mejorando las habilidades para detectar y analizar posibles problemas o ataques en una red. Se busca adquirir conocimientos prácticos sobre el análisis de paquetes y la interpretación de datos de tráfico de red.

Instalación y Configuración de Ubuntu Server

Administración de Sistemas Operativos

Documentación detallada sobre la instalación, configuración y administración de Ubuntu Server. Proyecto desarrollado como parte del curso Administración de Sistemas Operativos.

Visualizar PDF:

Descripción:

Este proyecto se centra en la instalación y configuración de Ubuntu Server, una distribución de Linux orientada a servidores. A lo largo del proyecto, se cubren los pasos para realizar la instalación desde cero, así como la configuración de servicios esenciales como SSH, servidores web, y bases de datos.

Además, se exploran aspectos como la administración de usuarios y permisos, la gestión de actualizaciones del sistema, y la implementación de medidas básicas de seguridad para asegurar el servidor.

Objetivos:

El objetivo principal de este proyecto es aprender a instalar, configurar y administrar un servidor basado en Ubuntu. A través de este proceso, se busca que los estudiantes adquieran habilidades clave para gestionar servidores Linux en entornos de producción.

Gestión de Usuarios y Grupos en Ubuntu Server

Administración de Sistemas Operativos

Documentación detallada sobre la administración de usuarios y grupos en Ubuntu Server. Proyecto desarrollado como parte del curso Administración de Sistemas Operativos.

Visualizar PDF:

Descripción:

Este proyecto aborda cómo gestionar usuarios y grupos en Ubuntu Server, cubriendo la creación, modificación y eliminación de usuarios y grupos. También se explica la configuración de permisos, así como el manejo de archivos importantes como /etc/passwd, /etc/group y /etc/gshadow.

La administración de usuarios y grupos es esencial en servidores Linux para controlar el acceso a recursos y mantener la seguridad del sistema.

Objetivos:

El objetivo principal de este proyecto es aprender a gestionar usuarios y grupos en Ubuntu Server, optimizando la seguridad y el control de acceso en un entorno de servidor Linux.
